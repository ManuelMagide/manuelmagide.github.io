## Proyectos destacados

### Fantasmin

Videojuego estilo *Vampire Survivors* desarrollado como proyecto académico.

Características principales:

* sistema de movimiento en tiempo real
* generación de enemigos
* sistema de combate automático
* progresión del jugador
* arquitectura modular orientada a objetos

Repositorio:
https://github.com/ManuelMagide/fantasmin

### Match-3 (Pygame)

Juego tipo Match-3 desarrollado en Python utilizando Pygame.

Características principales:

* grilla interactiva
* detección automática de combinaciones
* eliminación de fichas
* generación dinámica de nuevas piezas
* sistema de puntaje

Repositorio:
https://github.com/ManuelMagide/juego_match-3
